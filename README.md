Desafio Itaú – API de Transações e Estatísticas

API REST construída em Java + Spring Boot, conforme especificações do desafio técnico do Itaú.

O objetivo é processar transações financeiras e calcular estatísticas em tempo real com base nas transações dos últimos 60 segundos.

Tecnologias Utilizadas:

-Java 17

-Spring Boot 3

-Maven

-Spring Web

-Jakarta Validation

-Collections Concurrent (ConcurrentLinkedDeque)

-DoubleSummaryStatistics para cálculo de estatísticas
